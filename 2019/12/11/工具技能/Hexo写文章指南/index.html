Hexo Blog文章撰写

  1. 在source/_posts文件夹下面建立 .md新文档。
  2. 使用的markdown是一种程序化的写文章语言，方便放代码块
  3. md语法参考文章markdown书写指南
  4. 插入图片最好的方法，在/_post下面建立同名文件夹hexo n mdName,将图片放入该文件夹下。
